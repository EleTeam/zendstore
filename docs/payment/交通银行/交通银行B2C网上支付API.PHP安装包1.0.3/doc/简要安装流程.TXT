#交通银行网上支付系统API.PHP.UNIX安装包 简要安装流程

1、安装java平台
2、新建目录bocommjava(可自定义)，并将安装包内ini、cert目录复制进去，由于是通过ssl连接，如果您要做订单提交之外的其它交易，请先测试一下443端口的访问权限，详细见文档
3、将demo配置在已安装的服务器下(请将demo里的配置文件路径换为您本机的配置文件路径)
4、导入证书（详细见文档）
5、启动JavaBridge（在demo\lib下)，注意启动的java平台应与证书导入的一致（可用全路径执行java命令以确保运行的一致性），详细启动过程见文档
6、如果提交的订单出现乱码，请查看两个地方的设置：(1)php.ini里的default_charset设置 (2)apache服务器的https.conf里的AddDefaultCharset的设置。如果两个地方的设置与脚本文件的编码设置不一致就会导致乱码
7、将 define("JAVA_DEBUG", true); 开放就进入了调试模式，返回的结果信息将会包括详细的java调用过程，如果您遇到问题，可开放此设置，并将返回的结果发给我们
8、如果出现返回的内容不能显示(如显示object id #2)，就可能是调用返回的java变量没有正确的转换为php的变量类型，可用java_values 进行处理就可以正确的显示了
   
提示：如果您在联调的过程中仍然有问题没有解决，请将bocommjava下的log目录和JavaBridge生成的VMBridge.log(如有必要，可将您写的相关代码也一起)打包通过邮件发给我们，我们会即时给您解决
